



# Saved - KMP Bookmarks App

Клиент-серверное приложение для сохранения, категоризации и умного анализа ссылок.

*Документация по бэкенду и настройке локальной LLM находится в [репозитории бэкенда](https://github.com/ufalltag/GoProject).*

## Реализованные фичи
1. **Аутентификация** (Регистрация / Вход, JWT-токены).
2. **Умное сохранение ссылок** (Анализ URL через локальную LLM, авто-определение папки).
3. **Управление папками** (Создание, переименование, удаление).
4. **Поиск** (Поиск по закладкам).
5. **Профиль и Настройки** (Управление аккаунтом, смена пароля, переключение светлой/темной темы).

## Стек технологий 
- **Язык:** Kotlin Multiplatform
- **Архитектура:** Clean Architecture + Orbit MVI
- **Сеть:** Ktor Client
- **DI:** Koin
- **Аналитика:** Firebase Analytics & Crashlytics
- Настроен **CI/CD** c линтером и сборкой проекта
